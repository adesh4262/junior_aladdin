SECTION 1: THE SIX LAWS 

Every decision in this system — every strategy, engine, line of code — must satisfy at least one law. If it violates any, it is rejected. 

Law 1 — Context First, Action Second. No trade exists in isolation. Before any chart is analyzed, the system must know the macro environment, the regime, the time of day, and the day type. A technically perfect setup in the wrong 
context is a loss waiting to happen. Professional traders reject 80% of "good" setups because the context is wrong. 

Law 2 — Confluence or Silence. A trade fires only when Technical Structure + Microstructure + Options Pressure + Narrative + Regime + Time Context all agree. If even one dimension is strongly against, the system goes silent. The 
best traders in the world are flat 70% of the time. 

Law 3 — Anti-Fragility. Every loss makes the system more cautious (smaller size, higher thresholds). Every win does NOT make it more aggressive. The system becomes stronger under stress. This is the opposite of human psychology 
— and that difference IS the edge. 

Law 4 — Trap Awareness. Before every trade, the system asks: "Is this a trap?" Markets are designed to take money from the majority. The Trap Detection Engine runs on EVERY signal before scoring. This is not optional — this is the 
single most important edge a systematic trader can have. 

Law 5 — Survival First, Profit Second. The system's primary job is to NOT lose money. Any day ending flat is a successful day. Any day avoiding a large loss is a victorious day. Profits come from consistency over months. 

Law 6 — Compliance First. No feature, no speed, no edge that violates SEBI's algo trading framework enters this system. Static IP, Algo-ID, Limit orders only, audit trail — all non-negotiable. A non-compliant system is no system.

Hard Rules: Max risk per trade: 0.5% of capital. Max daily loss: 2% → LOCKED. Max trades/day: 5. Max consecutive losses before pause: 3. 

Practical ₹50K Capital Math (from Plan 2): 1 lot = 65 quantities. ATM premium ~₹50-100, margin ~₹10K-15K. Risk per lot with ₹15 SL = 65×15 = ₹975 (1.95% — close to daily max). Reality: 1 lot with ₹3-4 SL on premium, or OTM 
options with ₹15-30 premium and ₹5 SL (risk = 65×5 = ₹325). 

Six Sizing Multipliers (sequential): GARCH (forecast >1.5× median → ×0.5). Drawdown (>5% → ×0.75, >8% → ×0.5, >10% → LOCKED). Tilt (>70 → ×0.5). Session (LUNCH/CLOSING → ×0.5). Expiry (weekly → ×0.7, monthly → ×0.5). 
Score (>80 → ×1.0, 70-80 → ×0.8, 60-70 → ×0.6). 

Final lots = max(1, floor(base × all multipliers)).

TASK 4.5 — Black-Scholes Module (30 min) 

CREATE FILE: src/utils/black_scholes.py 

PURPOSE: Compute IV and Greeks for options when broker doesn't provide them. 

         Angel One does NOT give IV/Greeks — we compute locally. 

FUNCTIONS: 

  black_scholes_price(S, K, T, r, sigma, option_type="CE") → float 

    - S = spot price, K = strike, T = time to expiry (years) 

    - r = risk-free rate (0.065 for India), sigma = volatility 

    - Returns theoretical option price 

  implied_volatility(market_price, S, K, T, r, option_type="CE") → float 

    - Newton-Raphson iteration 

    - Start guess: sigma = 0.15 (15%) 

    - Iterate: adjust sigma until BS_price matches market_price within ₹0.10 

    - Max 50 iterations 

    - If no convergence: return last sigma estimate 

    - Return sigma (e.g., 0.13 = 13% IV) 

  compute_delta(S, K, T, r, sigma, option_type) → float 

  compute_gamma(S, K, T, r, sigma) → float 

  compute_theta(S, K, T, r, sigma, option_type) → float 

  compute_vega(S, K, T, r, sigma) → float 

  compute_all_greeks(S, K, T, r, sigma, option_type) → dict 

    Returns: {"delta": x, "gamma": x, "theta": x, "vega": x}

CREATE FILE: src/core/auth_manager.py 

PURPOSE: Handle Angel One authentication, token management, auto-refresh. 

CLASS: AuthManager 

ATTRIBUTES: 

  smart_api: SmartConnect object 

  auth_token: str (JWT token for API calls) 

  feed_token: str (for WebSocket) 

  refresh_token: str 

  last_auth_time: datetime 

  is_authenticated: bool 

METHODS: 

  authenticate() → bool: 

    1. Load credentials from .env using python-dotenv 

    2. Create SmartConnect(api_key) 

    3. Generate TOTP using pyotp.TOTP(secret).now() 

    4. Call generateSession(client_id, password, totp) 

    5. If success: store all tokens, set is_authenticated=True 

    6. If fail: log error, return False 

    7. Log: "Authenticated successfully" with client_id 

  refresh_session() → bool: 

    1. Check if token is older than 25 minutes 

    2. If yes: call generateToken(refresh_token) 

    3. Update auth_token 

    4. Log: "Token refreshed" 

  get_smart_api() → SmartConnect: 

    If not authenticated: call authenticate() 

    If token old: call refresh_session() 

    Return smart_api object 

  logout(): 

    Call smart_api.terminateSession(client_id) 

    Set is_authenticated = False 

ERROR HANDLING: 

  - Network error → retry 3 times with 2-second gap 

  - Invalid credentials → log CRITICAL, do not retry 

  - TOTP expired → generate new TOTP, retry once 

  - Session expired → full re-authenticate

TASK 5.2 — Instrument Token Mapper (Day 5, 2 hours) 

CREATE FILE: src/core/instrument_mapper.py 

PURPOSE: Map NIFTY option symbols to Angel One instrument tokens. 

         Example: NIFTY 24500 CE 01APR2026 → token "51234" 

BACKGROUND: Angel One uses numeric "tokens" to identify instruments. 

You need to know the token to subscribe via WebSocket or place orders. 

These tokens change with each expiry — so we build the map at startup. 

CLASS: InstrumentMapper 

METHODS: 

  build_map(smart_api, spot_price) → dict: 

    1. Call smart_api.searchScrip("NFO", "NIFTY") → returns ALL NIFTY options 

    2. Parse each result: 

       - tradingsymbol: "NIFTY01APR2526400CE" 

       - Extract: expiry date, strike price, option type (CE/PE)

    1. Parse message — Angel One sends binary/JSON with fields:

CREATE FILE: src/core/tick_validator.py 

CLASS: TickValidator 

STATE: 

  last_timestamp: datetime = None 

  last_price: float = None 

  stats: {total: 0, rejected: 0, spikes: 0, duplicates: 0, stale: 0} 

METHOD: validate(tick: dict) → (is_valid: bool, flags: dict) 

  RULE 1 — TIMESTAMP ORDER: 

    if tick.timestamp <= self.last_timestamp: 

      stats['duplicates'] += 1 

      return (False, {"reason": "out_of_order_timestamp"}) 

  RULE 2 — PRICE CONTINUITY: 

    if self.last_price is not None: 

      change_pct = abs(tick.price - self.last_price) / self.last_price 

      if change_pct > 0.02:  # 2% change in one tick 

        flags['spike'] = True 

        stats['spikes'] += 1 

        # Do NOT reject — legitimate gap openings exist 

        # But flag it for Feature Engine to suppress patterns 

  RULE 3 — MARKET HOURS: 

    tick_time = tick.timestamp.time() 

    if not (time(9,15) <= tick_time <= time(15,30)): 

      if time(9,0) <= tick_time < time(9,8): 

        flags['pre_auction'] = True  # Store separately for gap analysis 

      return (False, {"reason": "outside_market_hours"}) 

  RULE 4 — PRICE SANITY: 

    if tick.price < 10000 or tick.price > 50000: 

      stats['rejected'] += 1 

      return (False, {"reason": "price_out_of_range"}) 

  RULE 5 — STALE DETECTION: 

    gap_ms = (tick.timestamp - self.last_timestamp).total_seconds() * 1000 

    if gap_ms > 5000: flags['stale'] = True 

    elif gap_ms > 3000: flags['delayed'] = True 

    elif gap_ms > 1000: flags['slow'] = True 

  # All checks passed 

  self.last_timestamp = tick.timestamp 

  self.last_price = tick.price 

  stats['total'] += 1 

  return (True, flags)

CLASS: CandleBuilder 

STATE: 

  candles: {"1min": deque(maxlen=400), "3min": deque(maxlen=140), 

            "5min": deque(maxlen=80), "15min": deque(maxlen=30)} 

  forming_1min: dict = None  # Current incomplete candle 

  closed_1min_buffer: list = []  # Buffer for aggregation 

  last_1min_close_time: datetime = None 

METHOD: on_tick(price: float, volume: int, timestamp: datetime): 

  """Called for every validated tick""" 

  current_minute = timestamp.replace(second=0, microsecond=0) 

  IF self.forming_1min is None: 

    # First tick of the day — start new candle 

    self.forming_1min = { 

      "timestamp": current_minute, 

      "open": price, "high": price, "low": price, "close": price, 

      "volume": volume, "is_complete": False 

    } 

    RETURN 

  IF current_minute > self.forming_1min["timestamp"]: 

    # Minute changed! Close the forming candle. 

    self.forming_1min["is_complete"] = True 

    self.forming_1min["close"] = self.forming_1min["close"]  # last tick's price 

    # Add to 1-min deque 

    self.candles["1min"].append(self.forming_1min.copy()) 

    # Add to buffer for higher TF aggregation 

    self.closed_1min_buffer.append(self.forming_1min.copy()) 

    # Check if higher timeframes need closing 

    self._check_higher_timeframes() 

    # Start new forming candle 

    self.forming_1min = { 

      "timestamp": current_minute, 

      "open": price, "high": price, "low": price, "close": price, 

      "volume": volume, "is_complete": False 

    } 

  ELSE: 

    # Same minute — update forming candle 

    self.forming_1min["high"] = max(self.forming_1min["high"], price) 

    self.forming_1min["low"] = min(self.forming_1min["low"], price) 

    self.forming_1min["close"] = price 

    self.forming_1min["volume"] += volume 

METHOD: _check_higher_timeframes(): 

  buffer = self.closed_1min_buffer 

  buffer_count = len(buffer) 

  # 3-minute candle: every 3 closed 1-min candles 

  # Alignment: 9:15-9:18, 9:18-9:21, etc. 

  if buffer_count >= 3 and buffer_count % 3 == 0: 

    last_3 = buffer[-3:] 

    candle_3m = self._aggregate(last_3) 

    self.candles["3min"].append(candle_3m) 

  # 5-minute candle: every 5 

  if buffer_count >= 5 and buffer_count % 5 == 0: 

    last_5 = buffer[-5:] 

    candle_5m = self._aggregate(last_5) 

    self.candles["5min"].append(candle_5m) 

  # 15-minute candle: every 15

    last_15 = buffer[-15:] 

    candle_15m = self._aggregate(last_15)

    "low": min(c["low"] for c in candle_list), 

    "close": candle_list[-1]["close"], 

    "volume": sum(c["volume"] for c in candle_list),

  """Called every 500ms""" 

  now = ist_now() 

  # Tick gap 

  if self.last_tick_time is None: 

    tick_gap_ms = 999999 

  else: 

    tick_gap_ms = (now - self.last_tick_time).total_seconds() * 1000 

  # Option chain age 

  if self.last_option_update is None:

  else: 

    option_age_sec = (now - self.last_option_update).total_seconds() 

  # Ticks per second

    window_sec = (self.ticks_in_window[-1] - self.ticks_in_window[0]).total_seconds() 

    tps = len(self.ticks_in_window) / max(window_sec, 0.001) 

  else: 

    tps = 0 

  # Health classification 

  if tick_gap_ms > 5000: 

    health = "DOWN" 

  elif tick_gap_ms > 3000: 

    health = "STALE" 

  elif tick_gap_ms > 1000: 

    health = "DELAYED" 

  else: 

    health = "HEALTHY" 

  # Data quality score (0-100) 

  score = 0 

  if health == "HEALTHY": score += 40 

  elif health == "DELAYED": score += 20 

  if option_age_sec < 60: score += 20 

  if tps > 1: score += 10 

  # Remaining 30 points added by Feature Engine (no spikes, TF consistent) 

  self.current_health = health 

  self.data_quality_score = score 

  return health, score

# JUNIOR ALADDIN AI TRADING TERMINAL — COMPLETE DASHBOARD VISUAL DESCRIPTION

## Overall Dashboard Style & Theme

The dashboard is designed as a premium futuristic institutional-grade AI trading terminal inspired by Bloomberg Terminal, Obsidian Night UI, and modern fintech interfaces. The complete interface uses a dark neon cyber-finance theme with deep navy blue and black backgrounds.

Main accent colors:

* Neon cyan / electric blue for borders, highlights, icons
* Bright green for bullish/profit/healthy states
* Red/orange for bearish/risk/warnings
* White/light gray for primary text
* Soft muted gray for secondary text

The complete dashboard feels:

* highly professional
* dense with information
* modular
* military-grade control system
* AI command center
* institutional trading cockpit

The UI should feel alive and real-time.

---

Everything uses:
